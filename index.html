Portfolio

Amazon Astro

I worked on Astro, joining the program early in its development through to the product's public launch and first customer ship.

My team was responsible for developing the software that controls the motion of the robot's head and body, and the behavior of the robot. Essentially anytime the robot moves, I can point and say "That's my code!"

Main Technologies: C/C++, Python, Robot Operating System (ROS), MATLAB, Docker, Git

Robot Navigation in Cluttered Environments with Deep Reinforcement Learning

The aim of this project, my master's thesis research, was to apply deep reinforcement learning to enable a mobile robot to navigate a cluttered room using cheap and accessible camera sensors. The video to the right is a demo of what the robot sees and it successfully avoiding obstacles.

I experimented with extending this model so that the robot could learn to navigate toward a target position while avoiding obstacles. I leveraged AWS to accelerate simulation and model training, and analyzed data to show that the my extended model worked to succesfully navigate different cluttered environments. The work is published here.

Main Technologies: Python, AWS EC2, MATLAB, Deep Learning, Robot Operating System (ROS)

Digital Video Streaming for Nano Quadcopters

The objective of this project was to create a digital video streaming platform for consumer quadcopters to address the demand in consumer drone market. My team (two other engineers) and I designed a custom board with a WiFi-enabled ARM MCU and mobile phone camera for the CrazyFlie nano-quadcopter platform.

My primary contributions were the design and firmware implementation of a UDP-based networking protocol for streaming compressed camera images, and a Java application (shown on the left) that receives and decompresses the images, and displays them to the user along with some other relevant metrics.

Main Technologies: Java, C, Embedded Systems, Networking, Image Compression

Mapping Robot

This robot maps out the floorplan of a room. It featured a custom designed and 3D printed LIDAR system, prototype hardware, and a python application to visualize LIDAR data.

In this video, you can see the floorplan of my apartment as the LIDAR sweeps around the room. The color of each point indicates the distance away from the sensor.

Main Technologies: C, Python, LIDAR, I2C, UART, Power Conversion, CAD
